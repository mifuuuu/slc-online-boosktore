how does it work? 
student pov:
1. picks an item and add to cart
2. click the card
3. checkout
4. enter student name and student id
5. choose payment options
6. get generated code
7. show the generated code to the staff/worker
8. get your items

staff
1. click orders
2. enter the generated code given by the student
3. process the payment
4. done

admin
1. can see the status of the shop
2. can add/edit/update inventory items
3. can add/edit/update staff/employees
4. generate code
